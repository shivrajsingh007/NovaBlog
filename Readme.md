# 📝 NovaBlog – Responsive Blog Website

NovaBlog is a modern and fully responsive blog website built using **HTML and CSS**.
The project focuses on clean UI design, responsive layouts, and a structured blogging interface that works smoothly across desktops, tablets, and mobile devices.

---

## 🚀 Features

* 📱 Fully Responsive Design
* 📰 Clean Blog Layout
* 🎨 Modern UI Styling
* 🧭 Navigation Bar
* 🖼 Featured Blog Section
* 📖 Structured Article Cards
* ⚡ Fast and Lightweight

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **Flexbox**
* **Responsive Design**

## 🎯 Purpose of the Project

The goal of this project is to demonstrate **frontend development skills** by creating a responsive blog layout that adapts to different screen sizes while maintaining a clean and readable design.

This project is ideal for showcasing **HTML and CSS layout skills** in a frontend portfolio.
